Cara Flash Firmware (Lengkap):

1. Colok kabel USB ke ESP32-C3 dan PC
2. Tahan tombol BOOT
3. Sambil tahan BOOT → tekan tombol RESET sekali
4. Lepas tombol RESET
5. Lepas tombol BOOT
→ ESP32-C3 sekarang masuk mode bootloader

6. Klik tombol Flash Firmware
7. Pilih port COM yang muncul
8. Checklist Erase all
9. Klik Install / Flash
10. Tunggu proses selesai (10–30 detik)
11. DONE — device akan restart otomatis
